Load and execute the hs-define-ui-spec skill for initializing or modifying the project's UI design system.

Read the skill definition at skills/hs-define-ui-spec/SKILL.md and follow its Process section step by step.

Steps:
1. Check if docs/ui-design.md already exists
2. Present options: adopt a pre-built style (`npx getdesign@latest list`) or create custom from template
3. Install the chosen style or generate custom design from template at skills/hs-define-ui-spec/references/design-md-template.md
4. Verify the design system and optionally generate a test component
